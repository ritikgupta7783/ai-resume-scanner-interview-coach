# AI Resume Scanner & Interview Coach

AI-powered platform that:

- ATS Resume Matching
- Job Description Analysis
- Skill Gap Detection
- Technical Interview Questions
- Behavioral Questions
- Personalized Roadmap
- Resume Generation

## Tech Stack

Frontend:
- React
- SCSS
- React Router

Backend:
- Node.js
- Express.js
- MongoDB
- Gemini AI

## Features

- Authentication
- Resume Upload
- ATS Match Score
- Skill Gap Analysis
- AI Roadmap Generation
- Resume PDF Generation

## Installation

Frontend

npm install
npm run dev

Backend

npm install
npm start
